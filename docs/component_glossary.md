# Component Glossary

## Core Views
*   **SplashScreen**: Initial loading screen.
*   **StartScreen**: Main menu with "Begin Journey" and mastery stats.
*   **NavBar**: Bottom navigation bar for global access to Home, Arcade, Shop, Skills, and Profile.
*   **QuizSetup**: Configures quiz parameters (though currently mostly automated).
*   **QuizView**: The main quiz interface. Handles question display, answering, and feedback.
*   **ResultsView**: Displays quiz results, XP gained, and options to retry or continue.
*   **SkillTreeView**: Visualizes player progress through different levels/themes.
*   **ShopView**: In-game store for purchasing items with coins.
*   **StickerBook**: Displays earned achievements/stickers.
*   **DailyLogin**: Modal for daily rewards.
*   **CertificateView**: Displays a shareable certificate of achievement.

## Common Components
*   **GameSummaryModal**: Generic modal for displaying game results (XP, coins) in arcade games.
*   **GameTutorialModal**: Generic modal for displaying "How to Play" instructions.

## Arcade / Minigames
*   **MinigameHub**: The arcade menu listing available minigames.
*   **DefinitionMatchGame**: Game 2 - Match words to definitions.
*   **LetterDeductionGame**: Game 3 - Hangman-style letter guessing.
*   **WordScrambleGame**: Game 4 - Unscramble letters to form words.
*   **WordLadderGame**: Game 5 - Transform one word to another by changing one letter at a time.
*   **WordSearchGame**: Game 1 - Find hidden words in a grid.
